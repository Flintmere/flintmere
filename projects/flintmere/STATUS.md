- **2026-04-25** (Coolify deploy attempt #6 — REAL root cause found): Curl install landed (no more "curl: not found" line). Migrations were a no-op this time ("No pending migrations to apply") and Next.js Ready in 1418ms. Yet all 3 healthcheck attempts got `wget: can't connect to remote host: Connection refused`. Container logs revealed the actual cause at startup line: `Local: http://8483d1f10424:3001` — Next.js bound to the **container hostname** (Docker auto-sets `HOSTNAME` env var to the container ID), not `0.0.0.0`.
