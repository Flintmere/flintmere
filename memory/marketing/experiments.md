# experiments.md

Append-only log of marketing experiments. Hypothesis, variant, metric, result. Most recent at the bottom.

Format:

```
### YYYY-MM-DD — <slug>

- Hypothesis: <what we expect and why>
- Variant: <what changed vs the control>
- Surface: <homepage hero | pricing page | blog header | social post | outreach subject line | …>
- Metric: <the one number that decides>
- Result: <added when known — keep, roll back, inconclusive>
```

Rules:

- One primary metric per experiment. Secondary metrics are observations, not results.
- Minimum observation window stated up front. No stopping early.
- Do not run two conflicting experiments on the same surface at the same time.
- Inconclusive is a valid outcome. Log it. Do not chase significance.
